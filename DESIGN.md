# 雪山登山紀錄 視覺設計規範 (DESIGN.md)

本影片嚴格遵循 `01-活動紀錄影片` 的視覺與設計系統，確保質感流暢優雅。

---

## 1. 畫面比例與解析度
- **解析度**：1920 × 1080 (16:9 橫式)
- **畫面格率**：30 FPS

---

## 2. 配色系統 (Color Palette)

| 顏色 | HEX | 用途 |
| :--- | :--- | :--- |
| **主背景色** | `#0A0A0A` | 影片背景、過渡黑場、開場與字幕底色 |
| **字卡主色** | `#F5EFE2` | 大字卡的主要文字顏色 (暖米白，比純白更柔和有質感) |
| **強調金色** | `#E8B14A` | 底部進度條、左上角章節編號、點擊播放按鈕等 |
| **字幕白** | `#FFFFFF` | 旁白口白字幕主要顏色 |
| **字幕背景** | `rgba(0, 0, 0, 0.6)` | 字幕背後的半透明圓角底盒 |

---

## 3. 字體與字級系統 (Typography)

| 元素 | 字體名稱 | 字重 | 大小 (1080p 比例) | 備註 |
| :--- | :--- | :--- | :--- | :--- |
| **大字卡** | `Noto Serif TC` | 900 (Bold) | `clamp(40px, 6vw, 92px)` | 畫面中心的靈魂字句，緩慢淡入淡出 |
| **口白字幕** | `Noto Sans TC` | 400 (Regular) | `clamp(18px, 2vw, 30px)` | 底部人聲字幕，字數 ≤ 25 字不換行 |
| **章節編號** | `JetBrains Mono` | 300 (Light) | 13px | 左上角指示元件，如 `01/12 — DAWN` |

---

## 4. 鏡頭運動與過場效果 (Camera Motion)
- **Ken Burns 效果**：每個分鏡的背景照片預設在該 beat 時長內，從 `scale(1.0)` 平滑線性放大到 `scale(1.08)`，增加動態臨場感。
- **邊緣暗角 (Vignette)**：添加上下暗角與四周暗角漸層，保證中間的大字卡與底部的字幕具備極高的易讀性。
- **過場動畫**：分鏡切換時採用平滑的 `opacity` 淡入淡出 (0.8s)，不做花哨的 Wipe、3D 或 Carousel 效果。

---

## 5. 控制元件 (UI Overlays)
- **開場點擊遮罩 (Start Screen)**：
  - 當偵測到網頁無 `?render=true` 參數時，顯示一全黑底、金邊播放鈕的點擊遮罩，點擊後播放音樂並開始計時。
  - 當 `?render=true` 存在時，遮罩在 HTML 加載完成時完全隱藏並直接開始播放，以便 Playwright 順暢錄影。
- **進度條 (Progress Bar)**：
  - 底部邊緣的一條 2px 高度細線，隨著時間推進自 `0%` 增長到 `100%`，顏色為金黃色 (`#E8B14A`)。
- **音訊淡出**：
  - 影片最後 10 秒 (第 50-60 秒) 音樂音量線性由 `1` 降至 `0`，最終畫面停留在最後一幀。
